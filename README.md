# Cards Workout

**Cards Workout** — это приложение для тренировок, основанное на случайном выборе карт. Каждая масть представляет определенное упражнение, что делает тренировочный процесс более разнообразным и увлекательным.

## Основные функции

- Случайный выбор карты, где каждая масть соответствует определенному упражнению.
- Возможность остановить и перезапустить тренировку.
- Подробное описание упражнений, которое открывается в виде всплывающего окна.

## Основные технологии

1. **UIKit**:
    - Основной фреймворк для создания пользовательского интерфейса. Используется для работы с представлениями, такими как карточки, кнопки и всплывающие окна.

2. **MVC (Model-View-Controller)**:
    - Архитектурный паттерн, используемый для структурирования приложения. Модель управляет данными, представление отвечает за отображение, а контроллер связывает их вместе.

3. **Auto Layout**:
    - Используется для создания адаптивного пользовательского интерфейса, который корректно отображается на всех устройствах.

## Ключевые особенности

- **Случайное переключение карт**: Каждое нажатие на экран вызывает появление новой карты с соответствующим упражнением.
- **Поддержка нескольких упражнений**: Каждая масть карт ассоциируется с определенным видом упражнения, что делает тренировку интересной и разнообразной.
- **Интерактивное всплывающее окно**: Предоставляет пользователю дополнительную информацию об упражнении при необходимости.
